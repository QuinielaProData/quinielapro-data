# MDI QuinielaPro Data Repository

Repositorio oficial de distribución de datos para MDI QuinielaPro.

## Principios

- GitHub es la fuente oficial de distribución.
- Android consume datos.
- Android no publica datos.
- Android no contiene tokens.
- Room es caché regenerable.
- PredictionEngine solo consume Room.
- El Viewer PC es el responsable de mantener y publicar los datos oficiales.

## Flujo principal

Combinación Ganadora
↓
Viewer PC
↓
GitHub
↓
Android
↓
Room
